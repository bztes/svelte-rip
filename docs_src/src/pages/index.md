# svelte-rip

Svelte **R**esponsive **I**mage **P**review

- Component to display BlurHash images
- Component to display responsive images

## Installation

Run this command:

```shell
$ npm i -D svelte-rip
```

Then you can import it in your projects, e.g.:

```html
<script>
  import { Image } from 'svelte-rip';
</script>
```
